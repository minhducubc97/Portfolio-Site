# Duc Nguyen - Portfolio Website

This is the source code for my personal portfolio website.

## Structure
- `index.html`: Main entry point.
- `style.css`: Cyber-Minimalist styles.
- `script.js`: Particle animations and interactive effects.

## How to Deploy to GitHub Pages

1.  **Create a Repository:** Create a new repository on GitHub (e.g., `portfolio`).
2.  **Push Code:** Push these files to the repository.
3.  **Enable Pages:**
    -   Go to **Settings** > **Pages**.
    -   Under **Source**, select `Deploy from a branch`.
    -   Select the `main` (or `master`) branch and the `/ (root)` folder.
    -   Click **Save**.
4.  **View:** GitHub will provide a link (usually `https://<username>.github.io/<repo-name>`).

## Custom Domain (Optional)
If you have a custom domain, you can configure it in the Pages settings as well.
